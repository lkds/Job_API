### 前端接口规范：
* 返回json说明
    ```JavaScript
    {
        status:状态码 1成功 0 失败,
        msg:提示信息，如“查询失败“，一般请求失败时解释原因,
        body:返回的数据，若无返回数据则为空
                针对每个API有不同的解释
    }
    ```
